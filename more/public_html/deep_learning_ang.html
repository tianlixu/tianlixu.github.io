2.1 Logistic Regression

  • Given x, want \(\hat{y}\) = P(y=1|x) x ∈ Rn,
  • Parameters: w ∈ Rn, b ∈ R
  • Output:

    \(\hat{y}\) = wTx + b, is a linear functon of input x

    \(\hat{y}\) = σ(wTx + b)

    σ(z) = \(\frac{1}{1+e^{-z}}\), or σ(z) = \(1\Big/(1+e^{-z})^{}\)

    • σ(0) = 0.5
    • σ(∞) = 1
    • σ(-∞) = 0

2.2 Cost function

  • \(\hat{y^i}\) = σ(wTxi + b), where σ(zi) = \(\frac{1}{1+e^{(-z)^{i}}}\)
  • Given {(x1, y1), …, (xm, ym)}, train the model and find w and b so that \(\hat{y}^i\) ≈ yi
  • Loss (error) function
    • want loss function as small as possible
    • First function would be the squared root error L(\(\hat{y}\), y) = \(\frac{1}{2}\) (\(\hat{y}\) - y)2
      • we do not use this because it my leads to optimization problem which is not convex, means it contains local optimum points.
    • The loss function we'll use: L(\(\hat{y}\), y) = -(ylog\(\hat{y}\) +(1-y)log(1-\(\hat{y}\)))
      • if y=1 L(\(\hat{y}\), y) = -log\(\hat{y}\), want log\(\hat{y}\) large, want \(\hat{y}\) large(y->1)
      • if y=0 L(\(\hat{y}\), y) = -log(1-\(\hat{y}\)), want log(1-\(\hat{y}\)) large, want \(\hat{y}\) small(y->0)
      • the loss functiion was defined with respect to a single training example
  • Cost function which measures how well you are doing on the entire training set.
    • J(w, b) = \(\frac{1}{m}\) \(\sum_{i=1}^{m}\) L(\(\hat{y}^{i}\), yi) = -\(\frac{1}{m}\) \(\sum_{i=1}^{m}\) [yilog\(\hat{y}^{i}\) + (1-yi)log(1-log\(\hat{y}^{i}\))]
