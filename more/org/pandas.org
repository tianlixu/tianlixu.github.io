* pd.Series
pd.Series is like a list, which represents a clolumn in a pd.DataFrame.
example:
pds = pd.Series([1, 2, 3])

>>> pds
0    1
1    2
2    3

It's 0-based indexing.
>>> pds[0]
pds[0]                                                                                                                       
1

* pd.DataFrame
pd.DataFrame is like a dict(a table in a database).
